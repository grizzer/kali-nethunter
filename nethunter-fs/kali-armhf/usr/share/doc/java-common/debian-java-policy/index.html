Abstract

This is the java policy for Debian. It begins with a background description, continues with the real policy, some issues to discuss and ends with some advices to java packagers.

The policy covers java virtual machines, java compilers, java programs and java libraries.

This policy is published under the GNU GPL v2 or later license.
